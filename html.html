What is HTML?

HTML (HyperText Markup Language) is the language used to create the structure of a webpage. Think of it as the skeleton of a website. It organizes content like text, images, links, and forms into a readable format.

Basic HTML Document Structure

Every HTML document follows a basic structure. Here's how it looks:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>This is a paragraph of text.</p>
  </body>
</html>

Let's break it down:

  • <!DOCTYPE html>: Declares the document as an HTML5 document.
  • <html>: The root element that wraps the entire document.
  • <head>: Contains meta-information about the page, like the title and external stylesheets.
  • <body>: The main content of the webpage.

HTML Tags

HTML elements are created using tags. These tags help define the structure and layout of the content.

Common Tags:

  • <h1> - <h6>: Headings from largest to smallest (h1 being the main title).
  • <p>: Paragraphs, used to define blocks of text.
  • <a>: Links to other webpages or resources.
  • <img>: Display images.
  • <ul>, <ol>: Unordered (bulleted) and ordered (numbered) lists, respectively.

Example:

<a href="https://www.example.com">Visit Example</a>

Here we create a link using the anchor tag <a> with the href attribute specifying the URL.

HTML Attributes

Attributes provide additional information about elements. They are added inside the opening tag of an element.

Common Attributes:

  • href: The URL of the link.
  • src: The source of an image or media file.
  • alt: Provides alternative text for images.
  • id: A unique identifier for an element.
  • class: A class name to apply styles to multiple elements.

Example of an image tag:

<img src="image.jpg" alt="A beautiful view">

Practice Challenge: Adding Audio to a Webpage

Now, let's create a webpage with audio! Follow the steps below to get started:

  • A heading (<h1>) that introduces your webpage
  • A paragraph (<p>) explaining what the audio is about
  • A list (<ul>) of instructions for using the audio player
  • A link (<a>) to your favorite music or a page related to the audio
  • An audio player (<audio>) that allows users to play a sound or song

Here's an example of how you can add an audio player:

<audio controls>
              <source src="your-audio-file.mp3" type="audio/mp3">
              Your browser does not support the audio element.
            </audio>

Explanation:

  • The <audio> element is used to embed sound content.
  • The controls attribute adds play, pause, and volume controls to the audio player.
  • The <source> element specifies the path to the audio file and the format of the file (MP3 in this case).
  • If the browser doesn't support the <audio> element, the text inside the element ("Your browser does not support the audio element.") will be displayed instead.

Now, try the challenge! Create a webpage with the following:

  • A heading with the title of your audio content
  • A paragraph explaining the audio (e.g., what it's about or where it's from)
  • A list of features or instructions for the audio (e.g., "Click play to listen!")
  • An audio player with a file that you want to play (make sure you have a valid audio file like MP3 or OGG)
  • A link to a music website or another audio file

Once you're done, your webpage should look something like this:

<html>
            <head>
              <title>My Audio Page</title>
            </head>
            <body>
              <h1>Listen to My Favorite Song</h1>
              <p>This is a song I really enjoy. Press play to listen!</p>
              <ul>
                <li>Click the play button to start the song</li>
                <li>Pause anytime to stop</li>
              </ul>
              <audio controls>
                <source src="path-to-your-audio-file.mp3" type="audio/mp3">
                Your browser does not support the audio element.
              </audio>
              <a href="https://www.example.com">Visit my favorite music site</a>
            </body>
            </html>

Try modifying this example to use your own audio file and customize the content!
