What is CSS?

CSS (Cascading Style Sheets) is used to style the appearance of your webpage. It controls the layout, colors, fonts, and overall look and feel of the page.

CSS Syntax

CSS is written as a set of rules, each consisting of a selector and a declaration block:

selector {
  property: value;
}

Example:

h1 {
  color: red;
  text-align: center;
}

This would make all <h1> elements red and center the text.

CSS Box Model

The box model defines the layout of an element, including its padding, borders, and margins. Hereâ€™s a breakdown:

  • Content: The actual content inside the element, such as text or images.
  • Padding: Space between the content and the border.
  • Border: Surrounds the element's content and padding.
  • Margin: Space outside the border, separating it from other elements.

Visualizing the Box Model:

div {
  margin: 20px;
  padding: 10px;
  border: 5px solid black;
}

This will add 20px of margin around the element, 10px of padding inside the element, and a 5px solid black border.

Advanced Layout with Flexbox and Grid

Flexbox

Flexbox is a layout model that helps you design responsive layouts with ease. It gives you more control over the alignment and distribution of elements inside a container.

div {
  display: flex;
  justify-content: space-between;
}

This will space the child elements evenly within the parent <div>.

Grid

CSS Grid Layout is a powerful system for creating complex layouts with rows and columns.

div {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}

This creates a three-column grid layout with equal-width columns.
