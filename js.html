What is JavaScript?

JavaScript is a programming language that enables interactive web features, such as dynamic content updates, form validation, animations, and more. It runs in the user's browser, which means it can directly manipulate HTML and CSS to create dynamic experiences.

Variables and Data Types

Variables hold data that can be used in your program. JavaScript supports several data types:

  • String: A sequence of characters, wrapped in quotes.
  • Number: Integer or floating-point numbers.
  • Boolean: A value of true or false.
  • Object: A collection of key-value pairs.
  • Array: A list of values.

Example:

let name = "John";
let age = 30;
let isStudent = true;

Functions

Functions are reusable blocks of code that can be called whenever needed. Functions can also accept inputs (parameters) and return outputs (values).

function greet(name) {
  return "Hello, " + name + "!";
}

let message = greet("Alice");
console.log(message);  // Output: "Hello, Alice!"

DOM Manipulation

The DOM (Document Object Model) represents the structure of your webpage in a way that JavaScript can interact with. JavaScript allows you to change the content, structure, and style of your page by manipulating the DOM.

Example:

document.getElementById("myButton").addEventListener("click", function() {
  alert("Button clicked!");
});

This code listens for a click event on a button and shows an alert message when the button is clicked.

Control Flow: Loops and Conditionals

JavaScript allows you to control the flow of your program with loops and conditional statements:

Conditionals:

if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}

Loops:

for (let i = 0; i < 5; i++) {
  console.log(i);
}
